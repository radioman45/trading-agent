# 의사결정 저널 (append only — 트레이딩 + IPO 공용)

두 하네스의 모든 최종 판정을 한 저널에 기록한다 — 전체 투자 관점의 단일 의사결정 이력이다. 기존 항목은 수정하지 않는다 — `결과:` 줄만 갱신 가능하다(파이프라인 시작 시 중간점검 또는 복기 확정, 수익률·벤치마크·알파 포함).

- **트레이딩 항목:** portfolio-manager가 기록(APPROVE/REVISE/REJECT). 포맷은 `.claude/skills/risk-gate/SKILL.md`의 저널 포맷.
- **IPO 항목:** ipo-analysis 오케스트레이터가 Phase 6에서 기록(BUY/HOLD/SELL). 포맷은 아래:

```markdown
## {YYYY-MM-DD} {회사}({티커}) — {BUY/HOLD/SELL} {확신도}% (IPO, 상장일 {YYYY-MM-DD})
- 판정/확신도: {결론} {N}% (= 확신도 정의 1줄 — 무엇의 확률인가)
- 공모 기준: 공모가 {가격} ({source_status}) / 가치 앵커 요지(바닥 vs 옵션 프리미엄 비중)
- 크럭스: {결정적 쟁점 1~2개}
- 핵심 논거: ① ② ③ (검증 통과분)
- 채택 진입 계획: 실행 게이트 판정 / 성향별 최대 비중·분할 구간 요지 / 진입 보류 게이트(비대칭비·EV) 결과 / 잠재 손실(최대 비중 × 하방 시나리오)
- 가설훼손 체크포인트: {지표+임계값+시점 요지}
- 잔여 리스크: {알고도 수용한 것}
- 결과: (미기록 — 복기 시 갱신)
```
