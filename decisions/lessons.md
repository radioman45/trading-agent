# 교훈 (lessons, append only — 트레이딩 + IPO 공용)

복기에서 추출된 일반화 가능한 교훈 — 트레이딩은 `trade-reflection`, IPO는 `ipo-reflection`(같은 골격). 두 하네스의 모든 판단 에이전트가 분석 시작 시 자기 `적용 대상`에 해당하는 항목을 읽고 반영한다.

`적용 대상` 표기: 트레이딩 — bull/bear(researcher) · research-manager · trader · risk · portfolio-manager / IPO — bull/bear(analyst) · data-collector · fact-checker · judge · execution · verdict-reviewer / 두 하네스 모두 해당하면 **전체** (예: "이벤트 직전 일괄 진입은 갭에 취약", "쏠림 극단에서의 추격 매수").
