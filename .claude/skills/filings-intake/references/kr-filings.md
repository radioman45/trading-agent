# 한국 공시 구조 매핑 (DART 사업/반기/분기보고서)

한국 DART 정기공시의 섹션 구조와 SSOT 추출 위치. 사업보고서(연차)·반기보고서·분기보고서는 상세도가 다르다(분기 < 반기 < 사업).

## 1. 문서 구조 (대제목 매핑)

DART 정기보고서 표준 목차:

| 영역 | 한국 섹션 | 미국 대응 |
|------|----------|----------|
| 회사 개요 | "I. 회사의 개요" | Item 1 Business |
| 사업 내용 | "II. 사업의 내용" (제품·시장특성·경쟁요소·생산판매) | Item 1 + 산업 서술 |
| 재무 | "III. 재무에 관한 사항" (요약재무·연결/별도 재무제표·주석) | Item 8 / Item 1 |
| 경영진단 | "이사의 경영진단 및 분석의견" (III 내 또는 별도) | Item 7 MD&A |
| 감사·지배구조 | "VI~VII. 이사회·감사제도 등" | 별도 |
| 주주·주식 | "주식의 총수·자기주식" 등 | Equity 주석 |

> 한국 분기보고서는 MD&A에 해당하는 "경영진단" 서술이 빈약하거나 생략될 수 있다. 그 경우 MD&A 렌즈는 "II. 사업의 내용"의 서술과 재무 주석으로 보완한다.

## 2. 재무제표 위치 및 계정 매핑

"III. 재무에 관한 사항" — **연결재무제표 우선**(별도는 보조). 분기/반기는 검토(미감사) 재무제표.

| SSOT 필드 | 한국 계정명 | 비고 |
|-----------|------------|------|
| revenue | "매출액" / "영업수익" | 손익계산서 |
| cost_of_goods_sold | "매출원가" | |
| gross_profit | "매출총이익" | |
| operating_income | "영업이익" | 한국 GAAP/K-IFRS 영업이익 정의 |
| net_income | "당기순이익" | 지배기업 소유주 귀속분 구분 |
| eps_diluted | "희석주당이익" | 원 단위 |
| total_assets | 재무상태표 "자산총계" | |
| cash_and_equivalents | "현금및현금성자산" | |
| inventories | "재고자산" | |
| accounts_receivable | "매출채권" / "매출채권및기타채권" | |
| total_debt | "단기차입금"+"유동성장기부채"+"사채"+"장기차입금" 합산 | 합산 항목 명시 |
| total_equity | "자본총계" | |
| operating_cash_flow | 현금흐름표 "영업활동현금흐름" | |
| capex | "유형자산의 취득" (투자활동) | |

## 3. 단위·기간 규칙

- **단위:** 한국 공시는 "단위: 원" / "단위: 백만원" / "단위: 천원"이 혼재한다 — 표마다 확인. 억원 표기 서술도 있음. 모두 **원 단위 정수**로 정규화.
- **기간:** 분기보고서는 "당분기 3개월"과 "누적"을 함께 게재. 반기는 "반기 누적". `period_label` 명시.
- **연결 vs 별도:** SSOT는 **연결 기준**을 기본으로 한다(`source`에 "연결" 명시). 연결이 없으면 별도, 그 사실을 명시.
- **회계기간:** 대부분 12월 결산(역년). 다르면 `filing.fiscal_period`에 명시.

## 4. 세그먼트·주석 위치

- **세그먼트:** "II. 사업의 내용"의 부문별 매출 + 연결 주석 "영업부문 정보"(부문별 매출·손익·자산), 지역별 매출, 주요 매출처.
- **위험·우발:** "II. 사업의 내용 - 위험요인"(있으면), 주석 "우발부채 및 약정사항", "특수관계자 거래", "보고기간후 사건".
- **주식·자사주:** "주식의 총수", "자기주식 취득·처분", 자본변동표.

## 5. 흔한 함정

- 연결과 별도를 섞지 말 것 — SSOT는 연결로 통일.
- 단위 혼동(백만원 vs 천원 vs 억원)이 한국 공시 최대 오류원. unit_raw에 표 머리글 그대로 기록.
- "지배기업 소유주지분 순이익"과 "당기순이익(연결 전체)" 구분.
- 분기 단독값과 누적값 혼동 주의.
- 영업이익 정의: 일부 회사 공정공시·IR과 K-IFRS 표시가 다를 수 있다 — SSOT는 **재무제표 표시값** 기준.
