# Trade Planning — 거래 계획 설계

분석은 방향을 주지만 돈은 실행에서 벌린다. 이 스킬은 "매수"라는 단어를 "언제, 얼마에, 얼마나, 어디서 자르고, 어디서 먹는가"로 변환하는 방법론이다. **순서가 본질이다: 손실 한도 → 손절 레벨 → 포지션 크기 → 진입 → 목표.** 수익 기대에서 출발한 계획은 비중 과다로 끝난다.

## 0. 방향 정합성 (선행 점검)

거래 계획은 `03_research_plan.md`의 방향을 **절대 거스르지 않는다**:
- 방향 "매수" → 매수 계획
- 방향 "관망" → 계획 = "진입 보류 + 진입을 정당화할 조건(가격/이벤트)" — 억지 매수 계획 금지
- 방향 "매도/비중축소" → 기보유 가정의 청산/축소 계획 + 신규 진입 금지 명시

## 1. 설계 산식 (성향별 공통)

### 1-1. 손절 레벨 (먼저)
- 기술적 무효화 지점(technical 보고서의 지지선·추세 무효화선)과 리서치 무효화 조건 중 **더 가까운(보수적인) 쪽**을 기본으로 한다.
- 손절 폭은 최소 1.5 ATR — 그보다 좁으면 정상 변동성에 털린다(technical 보고서의 `atr14` 사용).
- 갭 리스크(실적 발표 등 이벤트 통과)가 있으면 "손절가가 보장되지 않음"을 명시한다.

### 1-2. 포지션 크기 (역산)
```
포지션 비중(%) = 1회 거래 허용 손실(계좌의 R%) ÷ 진입가-손절가 거리(%)
```
- R(1회 허용 손실): 보수 0.5~1% / 중립 1~1.5% / 공격 1.5~2%. 계좌의 2% 초과 금지.
- 산출된 비중이 성향별 상한(보수 5% / 중립 10% / 공격 15%)을 넘으면 상한으로 자르고 그 사실을 명시한다.

### 1-3. 손익비 (게이트)
```
손익비(R:R) = (목표가 - 진입가) ÷ (진입가 - 손절가)
```
- **1.5 미만이면 계획을 내지 않는다** — "손익비 미달, 보류 + 어떤 가격이면 성립하는지"가 결론이다. 억지 계획이 최악의 산출물이다.

### 1-4. 진입 전략
- 일괄 진입은 확신도 높음 + 이벤트 캘린더 무풍일 때만. 기본은 2~3회 분할(현재가 부근 / 지지선 부근).
- news 보고서의 이벤트 캘린더와 대조: 고변동 이벤트(실적·규제 결정) D-5 이내 신규 진입은 갭 리스크를 명시하고 분할 또는 이벤트 통과 후로 설계.

### 1-5. 목표가·시간 지평
- 목표가는 technical 저항 레벨과 리서치 논거(밸류에이션 회복, 촉매)에 닻을 내린다 — 희망 숫자 금지.
- 시간 지평(스윙 수주 / 포지션 수개월)을 명시하고, "기간 내 논거 미실현 시 청산 검토" 규칙을 넣는다.

## 2. 한국 투자자 비용 (의무)

- **미국 주식:** 환율(스냅샷 `fx.USDKRW`) 기준 원화 환산 명시. 양도소득세: 연 250만원 공제 후 22% — 목표 수익에 세후 관점 1줄. 환헤지 여부는 가정으로 표면화.
- **한국 주식:** 증권거래세(매도 시), 배당소득세. 대주주 요건은 일반 투자자 가정으로 생략 가능(가정 명시).
- 거래 수수료·슬리피지: 유동성 낮은 종목(거래량 보고서 참조)이면 슬리피지 가정을 명시.

사용자 성향이 입력에 없으면 **보수/중립/공격 3분기 모두** 작성한다:

```markdown

## 공통 전제
- 무효화 조건(리서치) → 철회 트리거(실행): {대응표}
- 이벤트 캘린더 요약: {진입 타이밍에 영향 주는 것만}
- 비용 가정: {환율·세금·수수료}

## {보수|중립|공격}형 계획   ← 성향별 반복
| 항목 | 값 | 근거 |
|------|-----|------|
| 진입 | {일괄/분할, 가격대} | {기술 레벨·이벤트} |
| 손절 | {가격} (-{X}%, {Y} ATR) | {무효화 지점} |
| 포지션 | 계좌의 {Z}% | R {r}% ÷ 손절거리 {X}% (상한 적용 여부) |
| 목표 | 1차 {가격} / 2차 {가격} | {저항·논거} |
| 손익비 | {R:R} | 1.5 이상 확인 |
| 시간 지평 | {기간} | {논거 실현 경로} |

## 모니터링 트리거
- 손절 외 철회: {이벤트·지표 기반 — 예: "다음 실적에서 마진 가이던스 하향 시 손절 전이라도 청산"}
- 비중 확대 조건: {있다면}
```

## 4. 검산 체크리스트 (저장 전)

- [ ] 방향이 리서치 계획과 일치하는가 (관망인데 매수 계획 아님)
- [ ] 모든 성향에서 손익비 ≥ 1.5 또는 "보류" 명시
- [ ] 포지션 산식 검산: R% ÷ 손절거리% = 비중, 상한 준수
- [ ] 손절 폭 ≥ 1.5 ATR
- [ ] 모든 가격 레벨에 근거(보고서·스냅샷 출처) 부착
- [ ] 무효화 조건 → 철회 트리거 변환 완료
- [ ] 환율·세금 가정 표면화
