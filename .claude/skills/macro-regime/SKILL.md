# Macro Regime — 톱다운 거시 레짐 판정

종목을 고르기 전에 **판의 국면**을 읽는다. 뉴로퓨전/월가아재의 핵심 명제: *"중요한 것은 단기 변동성이 아니라, 시장이 저점을 높이는 구조인지 고점을 낮추는 구조인지를 읽어내는 통찰"*이다. 좋은 섹터·종목을 골라도 투자 환경이 불리하면 무력하다 — 레짐 판정이 종목 선택보다 **상위 게이트**다.

## 판정 워크플로우 (7단계 스캔)

매크로 월간 레포트의 목차 구조를 표준 스캔 순서로 사용한다. 각 단계는 ▲(우호)/◆(중립)/▼(불리)로 채점하며 종합한다.

```
0. 핵심 질문   : 저점 높이는 구조 vs 고점 낮추는 구조? (단기반락 ≠ 레짐 전환)
1. 지정학      : 분쟁의 충돌/협력 병행, 이미 경험한 이벤트는 영향 희석
2. 물가·성장   : [물가] 기저압력 3변수 + 왜곡제거 + 5y5y 앵커
                [성장] 3섹터 순환 + PMI·투자·고용·소비 4축 + 하드/소프트 대조
3. 유동성·연준 : 대차대조표 항등식 5변수(SOMA·대출·RRP·TGA) + 발작 조기경보
                + 금리상승 3분해(밸류/성장/정책견제) + 재정우위(재무부>연준)
4. 달러·질서   : 미란 시퀀스(관세→다자환율→일방), 강→약달러 전환, 금·대체자산
5. 밸류·위치   : 시장 밸류 국면(Index DCF 내재기대·할인율 게이트) + LPPL 버블 Confidence
                + 사이클 위치(버블 후반 경계), 수급(옵션 쏠림·CTA)
6. 자산배분    : 내러티브 중첩 + 경기사이클 섹터회전 + 바벨 기울기
7. 종합·분지도 : 레짐 라벨 + 다음 분기 확률 + 무효화 조건
```

### 단계별 핵심 판정 규칙

- **성장축:** 3섹터 순환(고용→소비→생산)과 PMI·투자·고용·소비 4축을 ▲◆▼ 채점. 소프트(심리·설문)는 하드(고용·소비·국채금리·하이일드·BEI)와 대조해 확정. 프론트로딩·양극화·버퍼 소진 보정. → 견조/혼조/둔화.
- **물가축:** 기저압력 3변수(잔여 관세·에너지 충격·AI 비용) + 표면 왜곡 제거(근원 ex-shelter, OER 보정). 장기 기대인플레(5y5y Forward)의 앵커링 유지 여부가 "연준 신뢰 훼손"의 핵심 게이지. → 디스인플레/안정/재가속.
- **사분면 특정:** 성장축 × 물가축 → **골디락스**(견조·안정, 위험자산 우호) / **리플레이션**(견조·상승, 원자재·에너지) / **스태그플레이션형 긴축**(둔화·끈적, 위험자산 불리·원자재 우위) / **디플레/리세션**(둔화·하락, 방어자산). 현재 위치 + 다음 분기 확률.
- **유동성 국면:** 항등식 5변수 합산 + RRP 버퍼 ON/OFF + 발작지표 점등 → 우호/중립/긴축. "금리 인하 = 유동성 증가 = 상승"을 의심한다(인하가 RRP버퍼 OFF·MBS/BTFP 조기상환으로 오히려 지준금을 줄이는 4경로).
- **금리·연준 오버레이:** 금리 상승을 3분해(밸류 압력 −/성장 개선 +/정책 견제 +)해 **배후 내러티브**를 판별. 성장 내러티브면 주가-금리 동행(+), 인플레/재정불안 내러티브면 음(−)+긴축 우려.
- **정치재정 오버레이:** 선거주기 경직(완화 기대 할인), 재정우위(연준 인상 제약), 정책 시퀀스(관세→환율). 친/비성장 정책 강약 조합으로 노드 확률 조정.
- **밸류·버블:** Index DCF로 시장 내재 기대수익률(Implied 할인율) 역산 → 8~9% 범위 게이트. LPPL Confidence는 **지수에만** 적용하고 단독 매도 신호로 쓰지 않는다(조기경보 only — 고Confidence 후에도 수개월 상승 가능).

## 한국 시장 분기

미국 레짐(연준·유동성·달러)이 글로벌 상수다. 한국 분석 시 추가로: ① 한국은행 금통위·기준금리 ② 원/달러 환율과 달러 익스포져(월가아재: 환율 고점 구간의 헤지 고민) ③ 외국인 수급·코스피/코스닥 정책(상장 구조 개편) ④ 미국 레짐의 한국 전이(반도체·수출 사이클). 한국 종목도 미국 거시 레짐을 배경으로 깔되 국내 변수를 오버레이한다.

## 주의 — 과적합 금지

방법론의 특정 시황 결론(예: "26년 상반기 긍정")을 복붙하지 않는다. 그건 작성 시점의 뷰다. 너는 **절차**(사분면 판정·항등식 마킹·내러티브 판별·역발상)를 새 시점 데이터에 재적용한다. 보고서에 없는 수치를 지어내지 않는다.
