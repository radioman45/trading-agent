# Filings Synthesis — 공시 종합 리포트

4렌즈가 같은 공시를 서로 다른 각도에서 해부했다. 이 스킬은 그 조각들을 **하나의 입체적 그림**으로 엮는 골격과 원칙을 담는다 — 이번 공시가 보여주는 회사의 현재 상태를, 숫자·언어·구성·위험을 통합해 균형 있게.

## 핵심 원칙

- **종합하되 날조하지 않는다.** 모든 문장은 4렌즈 보고서·SSOT에 근거를 둔다. 렌즈에 없던 새 수치·새 주장을 만들지 않는다. 끌어온 핵심 수치는 SSOT/렌즈로 역추적 가능해야 한다.
- **일치와 상충을 둘 다 드러낸다.** 렌즈가 수렴하면(예: 재무 마진개선 + MD&A 믹스강조 + 세그먼트 고마진부문 성장) 그 신호의 강함을 말한다. 엇갈리면(예: MD&A 낙관 vs 리스크 신규소송) 그 긴장을 숨기지 말고 나란히. **상충이 가장 중요한 정보다.**
- **균형 의무.** 긍정·우려를 양쪽 모두. 한쪽으로 기운 리포트는 이해가 아니라 설득이다.
- **이해 리포트의 경계.** 결론은 "사라/팔라"가 아니다. "이번 공시가 말하는 것"으로 닫는다. 투자 판단이 필요하면 트레이딩 하네스로 가야 함을 1줄 안내.
- **확신도 대신 근거 강도.** 각 핵심 판단에 `공시로 확실 / 부분 근거 / 공시 외 확인 필요`를 표기.

## 리포트 골격

`_workspace_filings/03_synthesis.md`:

```markdown
# 공시·펀더멘털 이해 리포트 — {회사명} ({공시유형} {회계기간})
- 분석 시점: {YYYY-MM-DD} | 공시: {source_file} | 데이터: 제공 공시 원문만

## 1. 한눈 요약 (경영진 브리프, 7줄 이내)
{이번 공시가 보여주는 회사의 현재 상태 — 무엇이 일어났고, 강점·주의점 핵심. 매매 권고 아님}

## 2. 이번 공시의 핵심 변화 (Top 3~5)
{비교기간 대비 가장 중요한 변화. 각 항목에 [근거 강도] 표기}

## 3. 렌즈별 통합 해부
### 3.1 재무 (수익성·현금·건전성)
### 3.2 경영진 서술 (MD&A — 설명 vs 사실)
### 3.3 사업 구성 (세그먼트 — 성장엔진·믹스)
### 3.4 위험·주석 (경고등)
### 3.5 산업 맥락  ← industry 보고서 있을 때만, 없으면 생략

## 4. 렌즈 간 일치 · 상충
| 주제 | 렌즈들이 말하는 것 | 수렴/상충 | 해석 |

## 5. 다음 공시 관전 포인트
{이번 공시로 답 못 한 질문, 다음 분기에 확인할 지표·이벤트}

## 6. 데이터 한계 · 면책
{공시 미수록, 렌즈 누락, 제공 파일 경계. + "본 리포트는 제공 공시의 이해를 돕는 자료이며 매수/매도 판단이 아니다 — 투자 판단은 트레이딩 하네스 또는 사용자 책임" 1줄}
```

## 작성 규칙

- §3 각 소절은 해당 렌즈 보고서를 **압축·통합**한다(복붙 금지, 핵심만). 렌즈가 누락됐으면 "X 렌즈 부재로 제한" 명시.
- §4 상충 표가 이 리포트의 차별점이다 — 최소 2~3개 주제에서 렌즈 교차를 본다.
- 렌즈가 "특이 없음"이라 한 곳을 위험으로 부풀리지 않는다 — 종합은 증폭이 아니라 통합.
- 분량 1500~2500단어(산업 레이어 포함 시 +500).
