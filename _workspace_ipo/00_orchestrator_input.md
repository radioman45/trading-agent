# 투자 분석 입력

- 대상: 삼성전자 (005930.KS / 우선주 005935.KS)
- 시장/거래소: KR / KRX (코스피)
- 분석 시점: 2026-06-10
- 모드: **2차 시장 (일반 모드)** — 신규상장 아님. 모든 스킬은 IPO 분기가 아니라 기본(2차 시장) 모드를 적용한다.
  - 거래가·거래 이력·과거 멀티플이 존재한다 → PER/PBR/EV-EBITDA, 역사적 밴드, 동종 비교 사용 가능.
  - IPO 전용 항목(공모가·발행신주·락업·greenshoe·free float 4필드·지수편입 선반영)은 해당 없음 — 대신 현재가·시총·발행주식수(보통주/우선주)·자사주·외국인 지분율·배당을 고정한다.
- 시드 요약: 없음 (시드 노트 미제공 — 1차 출처에서 직접 수집)
- 사용자 맥락: 일반 분석 (성향 미지정 → 보수/중립/공격 3성향 모두 산출)
- 데이터 소스 우선순위: DART(전자공시) 사업보고서·분기보고서 > KRX 시세 > IR 자료 > 보도. 환율(원/달러)은 분석 시점 기준 명시.
- 핵심 사업 축(분석 참고): DS(메모리 DRAM/NAND·파운드리·시스템LSI), DX(스마트폰 MX·가전), SDC(디스플레이), Harman. HBM/AI 메모리 사이클, 파운드리 경쟁력(vs TSMC), 주주환원 정책이 주요 쟁점.
